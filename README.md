# tnt_scanner

A Barcode Scanner Application for TNT

## Getting Started to Test the Device

1. Save the barcode excel file in the root folder of the device. Example is This PC\EDA50\Internal shared storage
2. Login details is available in the password sheet of the excel file.
3. Login as "admin" and "admin".
4. There is an option for Arabic Translation in the login page.
5. Once signed in, first upload the database in the Database Tab Page.
6. Looking in the Main Database, this are the barcodes need to be scan.
7. In the Settings tab page, you can configure the application in terms of langauge, theme, sound and keyboard.
8. In the Scan Code tab page, this is where you can the code. The barcode should end with a "," symbol as the end of line.
9. For testing purposes, you can enable the keyboard to manually type a number and press "," to scan.
10. All scanned barcode will be displayed in the History tab page with its respected status.
11. You can select multiple scanned barcode and export it to the barcode excel.
12. Exported barcode are available in the output sheet of the excel file
